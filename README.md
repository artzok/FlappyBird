# FlappyBird
该项目利用LibGDX游戏框架重建了著名动作类游戏Flappy Bird，还原度100%，并且增加了个性化切换效果。该项目包含了桌面版和Android版，其中桌面版可以适配Windows、Mac OS、Linux等操作系统，如果希望在IOS上运行该游戏，可以利用Robvm重新编译，再导出安装包即可。
